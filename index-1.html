THE GOSSIP PROJECT : ON TRACK !

The Hacking Project - Bootcamp Fullstack Web

Semaine 6, jour 1.

On commence à lier les pages entre elles via Rails.

L'app affiche quelques pages statiques, et un peu dynamiques aussi.

Fonctionnalités présentes

  • affichage des pages 'Team' et 'Contact' en mode moche
  • affichage d'un header sur toutes les pages
  • affichage d'une page de bienvenue dynamique
  • affichage de la page de liste des potins
  • affichage de la page de détail d'un potin, possibilité d'aller voir la page de son user
  • affichage de la page de détail d'un user, possibilité d'aller voir ses potins

Installation

Cloner le repository : ~~~bash $ git clone https://github.com/gaael/gossip-project-v1.git ~~~ Ou télécharger sa version compressée puis l'extraire.

Se placer dans le dossier gossip-project-v1-master. Installer les gemmes nécessaires (via Bundler) ~~~bash $ cd path/to/gossip-project-v1-master $ bundle install ~~~

Réinitialiser la BDD\ /!\ fermer toute rails console et autre rails server ouverts avant cette opération, sans quoi Rails va râler... ~~~bash $ rails db:purge db:migrate db:seed ~~~

Remplir la BDD à partir de seeds.rb ~~~bash $ rails db:seed ~~~

Lancer le serveur rails ~~~bash $ rails server ~~~

Visiter le site en local :\ localhost sur le port 3000

Configuration requise

Version de Rails : ~~~bash $ rails-v Rails 5.2.3 ~~~

Version de Ruby : ~~~bash $ ruby -v ruby 2.5.1 ~~~

Version de Bundler : ~~~bash $ bundle -v Bundler version 2.0.2 ~~~

Gemmes utilisées (source : Gemfile) :\ Gemmes ajoutées au Gemfile de Rails : ~~~ruby group :added_gems do # Enables 'realistic' and 'effortless' seed gem 'faker' # Displays nice tables in Rails Console gem 'table_print' end ~~~
